# À la racine du projet
cd server && npm install      # installer serveur
cd ../client && npm install   # installer client

# Lancer les deux services (dans deux onglets)
cd server && npm start
cd client && npm start

# Pour le build de production (client)
cd client && npm run build



Optionnel Lancer les deux en une seule commande
Si tu veux tout faire d’un coup dans le même dossier racine, installe un outil comme concurrently :
# À la racine du projet
npm install --save-dev concurrently

dans package.json
{
  "scripts": {
    "dev": "concurrently \"cd server && npm start\" \"cd client && npm start\""
  },
  "devDependencies": {
    "concurrently": "^8.0.0"
  }
}

npm run dev


Ctrl + C = stoppe npm start.

Ouvre deux terminaux (ou onglets) pour garder le serveur et le client actifs en même temps.

Si tu préfères, utilise concurrently pour un lancer unique.
